---

### **PATH B: Auto-Deploy on Every Push (10 minutes)**

**Step 1: Deploy once manually (see PATH A above)**

**Step 2: Get GitHub Secrets**

Go to your **GitHub repo → Settings → Secrets and variables → Actions**

Add these 3 secrets from your Vercel project settings:
- `VERCEL_TOKEN` (from https://vercel.com/account/tokens)
- `VERCEL_ORG_ID` (from https://vercel.com/account/settings)
- `VERCEL_PROJECT_ID` (from your project settings)

**Step 3: Done!** 

Now every time you:
```bash
git push origin main
```

Your site **automatically updates**! No manual steps! 🚀
