# 🚀 **Deploy Your Zorvyn Dashboard - Live in 5 Minutes!**

## 📋 **Option 1: Deploy to Vercel (RECOMMENDED - Easiest)**

Vercel is the creator of Next.js and makes deployment super easy.

### **Step 1: Create Vercel Account**
1. Go to: https://vercel.com/signup
2. Click "Continue with GitHub"
3. Authorize Vercel to access your GitHub account
4. Done! ✅

### **Step 2: Import Your Project**
1. On Vercel dashboard, click "Add New..." → "Project"
2. Select "GitHub" and search for "Zorvyn-Assessment"
3. Click "Import"
4. Settings should auto-detect (Next.js)
5. Click "Deploy" 🚀

### **Step 3: Done!**
- Vercel gives you a free live URL: `your-project.vercel.app`
- Every time you push to GitHub, it auto-deploys! 🎉
- You get a preview link for pull requests too

**Cost**: FREE tier available (perfect for portfolio projects)

## 📋 **Option 2: Deploy to Netlify (Also Easy)**

### **Step 1: Create Netlify Account**
1. Go to: https://app.netlify.com/signup
2. Click "Sign up with GitHub"
3. Authorize and confirm email

### **Step 2: Build & Connect**
1. Click "Add new site" → "Import an existing project"
2. Choose GitHub and select "Zorvyn-Assessment"
3. Netlify auto-detects the build settings
4. Configure:
   - Build command: `npm run build`
   - Publish directory: `.next`
5. Click "Deploy site"

### **Step 3: Done!**
- Get your free URL: `your-site.netlify.app`
- Auto-deploys on every GitHub push
- Free HTTPS certificate included

**Cost**: FREE tier available
